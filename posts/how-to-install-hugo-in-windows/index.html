How to Install Hugo in Windows

In this article, Let’s learn about Installing Hugo in a Windows machine. It’s a straightforward step to install Hugo.

For the Linux machine, it’s just a piece of command.

sudo apt-get install hugo -y

Enjoy Linux User. For Mac OS, see the official documentation.

So, Let’s install Hugo in windows.

Go to Hugo Releases and download window binary. Extract the file where ever you want. Then copy the file path.

After that, Search for “Edit the system environment variables.” At the ground, click on the “Environment Variable” option. Then, In User Variable search for Path and then edit.

Paste the Hugo executable file path then save all the settings.

Now you are good to go.

Run Hugo in command prompt.
